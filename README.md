## Usage(recommended)
```bash
bash <(curl -f -L -sS https://raw.githubusercontent.com/abcfyk/impatriot/master/v2ray/ws_nginx_tls_1.3.sh)
```

## Note
- 切勿在安装了$$的服务器上使用此脚本，切记切记
- 推荐使用tls1.3脚本进行安装，安全性更好。但某些手机客户端的v2ray内核如果低于v4.18.1 可能不支持，请自行选择。
